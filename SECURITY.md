# Security policy

### Supported versions

Upgrade to new versions when they are available: Security patches for older versions will not normally be issued.

### How to report an issue

To report a serious issue, please follow
_[report a vulnerability](https://github.com/dmyersturnbull/science-notebook-template/security/advisories/new)_.

Minor security issues can alternatively be reported using the
[security issue template](https://github.com/dmyersturnbull/science-notebook-template/issues/new?labels=kind%3A+security+%F0%9F%94%92&template=security.md).
